# Laleo 🗣️

Aplicativo open source para ajudar crianças com dificuldades de fala: exercícios de fonoaudiologia gamificados, guiados por um avatar 3D interativo com IA. Web + Android + iOS.

*Laleo* vem do grego **λαλέω** (*laleō*) — falar, tagarelar.

## Estrutura

- `backend/` — API Java 21 / Spring Boot 3
- `frontend/` — React + TypeScript + three.js (avatar VRM) + Capacitor
- `speech-service/` — análise de pronúncia (transcrição + nota por fonema, pt-BR)
- `docs/` — arquitetura e decisões

Veja [docs/arquitetura.md](docs/arquitetura.md) para a visão completa.
